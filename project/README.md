## ✨ Features

- **Multiple Workspaces**  
  Create multiple workspaces for different teams or organizations.

- **Project Management**  
  Create and manage projects inside each workspace.

- **Task Management**  
  Assign tasks to team members, set due dates, and track progress.

- **Team Collaboration**  
  Invite members and manage their roles within the workspace.

- **Analytics Dashboard**  
  View project progress, completion rates, and team activity.

- **Responsive UI**  
  Clean and responsive interface built with Tailwind CSS.
